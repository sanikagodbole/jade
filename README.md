## Jade - A tool to streamline VFX Pipeline

### Introduction
This tool is use to streamline the creation of the folder structure for pipeline creation, update and versioning.

---

### Setup
You need to set the following environment variables in order to use this tool.

1. JADE_SHOW_NAME: the name of the show for which you are creating the pipeline.

2. JADE_COLLAB_BASE_DIR=: some root folder where the pipeline directories are created.

`export JADE_SHOW_NAME=Stone Lions Dont Roar`

`export JADE_COLLAB_BASE_DIR=/I-Drive/Savannah/CollaborativeSpace/stonelions/$JADE_SHOW_NAME/pipeline`
